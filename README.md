# FIT FOCUS

# Introduccion

"¿No tienes tiempo para ir al gimnasio? ¿No sabes cómo comenzar tu vida fit?
FIT FOCUS® es ideal para ti, esta es una aplicacion grupal desarrolladas por estudiantes d HENRY como primera gran aplicacion en la que aplicamos todas las tecnologias aprendidas en el bootcamp , entre otras mas.

# Objetivos del proyecto

- Desarrollar una aplicación con typescript desde cero
- Reafirmar y connectar todos los conocimientos aprendidos en la carrera
- Aprender y practicar GIT flow y trabajo en equipo
- Usar la metodologia Scrum

# Tecnologías usadas

# Front End:

HTML, Javascript, Typescript, React JS, Redux toolkit, CSS, TailwindCSS, Styled-Components.

# Back End:

Mongo DB, Mongoose, Mongo Atlas, Node JS, Json Web Token authentication, Nodemailer, oAuth.

# Instrucciones de incio

<b>IMPORTANTE:</b> Es necesario tener las siguientes versiones de node y NPM

- <b>Node:</b> 12.18.3 or higher
- <b>NPM:</b> 6.14.16 or higher

# Instalar los paquetes necesarios para iniciar

- Abrir la consola en el proyecto
  - En la ruta <b>api</b>, ejecutar el comando <b>npm install</b>
  - En la ruta <b>cliente</b>, ejecutar el comando <b>npm install</b>
  - Por ultimo, en la ruta <b>cliente<b>, ejecutar <b>npm start</b>

# Para probar el modo PREMIUM deberá usar los daots de tarjeta de prueba

    - Numero de tarjeta: 5031 7557 3453 0604
    - Fecha de vencimiento: 11/25
    - Codigo de seguridad: 123
    - En los otros datos puedes colocar lo que desée
